Project Overview
ShiftTrack is a cloud-hosted shift management system built for ICT171 at Murdoch University. Workers can clock in and out, view their shift history, and calculate total hours вЂ” all served from a manually configured Apache web server on a Microsoft Azure Ubuntu VM.
This project uses Infrastructure as a Service (IaaS). The virtual machine was provisioned on Azure and then configured entirely from scratch over SSH. No pre-installed marketplace bundles, SaaS website builders, or pre-configured images were used. Every step вЂ” Apache installation, DNS configuration, SSL/TLS setup, and script deployment вЂ” was carried out manually through the terminal.

Note on data storage: Shift records are stored in the browser's localStorage API. This is a client-side prototype вЂ” data is local to each browser and is not held in a central server database. A future version could use a backend database such as SQLite or MySQL.
