## Project Overview

ShiftTrack is a cloud-hosted shift clock-in and clock-out tracking system. The website allows a worker to record clock-in and clock-out times, view shift history, and calculate total hours worked.

This project was deployed on a manually configured Azure Ubuntu virtual machine using Infrastructure as a Service. Apache was installed and configured manually through SSH. The project also includes a Bash script that generates a server report page.

## Technology Used

- Microsoft Azure Virtual Machine
- Ubuntu 24.04 LTS
- Apache 2
- HTML
- CSS
- JavaScript
- Bash scripting
- GitHub documentation

## Main Features

- Clock in and clock out
- Worker name input
- Live date and time display
- Shift history table
- Total hours calculation
- Browser localStorage for saved records
- Server report generated by Bash script
